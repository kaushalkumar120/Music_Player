# 🎵 Advanced Music Player

A modern web-based **Music Player** built using **HTML, CSS, and JavaScript**. This project allows users to play songs directly in the browser with an interactive interface and smooth controls.

## 🚀 Features

* ▶️ Play / Pause music
* ⏭️ Next / Previous song controls
* 📊 Dynamic progress bar
* 🔊 Volume control slider
* 📃 Clickable playlist
* 🌙 Dark / Light mode toggle
* 💿 Rotating album cover while playing

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Audio API)

 controls, and responsive design.


Developed as a front-end practice project for learning modern web development.
